# Estrutura de Imagens Estáticas

Esta pasta concentra os assets estáticos do projeto.

## Pastas

- `logos/`
  - arquivos de marca do Visite Lapa
- `hero/`
  - fundos, texturas e elementos gráficos para áreas hero
- `placeholders/`
  - imagens auxiliares e placeholders para futuras telas

## Convenção

- usar nomes em minúsculas
- separar palavras com hífen
- preferir `svg` para logos, texturas e fundos leves
- manter caminhos públicos começando por `/imagens/...`
